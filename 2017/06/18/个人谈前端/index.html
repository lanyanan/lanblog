个人谈前端

不知道从何说起

说前端那就先看看与前端相关的东西的发展：第一个是浏览器的变化，最近几年的浏览器开始越来越多的支持h5和css3 的api。 第二个变化是手机的变化，手机变的越来越快对于h5的性能的提升显而易见，还有就是手机的使用变得越来越多，电脑的使用变的越来越少。第三个变化是关于微信的，微信公众号的推出催生了很多的产业，也使得h5更加的流行，再加上h5小程序的推出以及版本的迭代，h5的开发会越来越流行。基于以上三点可以看到市场上的需求越来越多，这就会推动前端越来越快的发展。

前端技术生态格局

现在的前端是框架的天下，现在的框架是三足鼎力 react angular vue 今年发展最火的非vue莫属有可能未来的方向都是vue为主，但是到目前为止react的生态圈是最完善的，而angualr2.0的出现竟然不能把1.0的升级到2.0，这也是最大是败笔，可能再过一段时间当vue的社区和生态逐渐完善之后就会vue位居第一位了。

大前端的形成

node 的出现以及v8引擎的出现使得js具备了做大型项目的能力，也就是说可以使用node来书写后台然后运行在v8引擎上面使得前端开发者可以迅速具有后台开发的能力，这样的话我们称之为大前端，也就是前后端一起来。当然使用node来做后台一般都是些创业公司用的，因为这样前后端一起来是可以节省开支的。

全站工程师

说到全站工程是大家可能想到的是会使用几门编程语言的人比如开发后台要会java php 开发android要会 java 开发ios的要会c++ 前端的要会js等等，但是现在你可以只会一种语言就可以通吃，那就是js 当然我们是要借助框架的比如说后台的我们会使用node 前端的我们会使用react框架 app的开发我们可能会使用react-native来开发 这样的话你就可以做网站 做app 这样的你应该可以成为全栈工程师

现在的格局

今年观看谷歌的io大会感触良多，看到了ai技术的便利看到ai技术的强大 可以通过一个例子来佐证我的 我们可以跟我们的智能音箱进行交互 让他帮我们查看天气状况，出行情况，以及可以让她把我们的家用电器开启和关闭，还能在电视上播放我们想要的视屏，那么相信未来可能我的家庭里面所有的智能设备都可以通过智能音箱来下发指令，不用使用遥控器，按钮，以及app。那么这个就悲剧了ai的出现势必会导致app开发数量的减少，进而导致需求的减少，那么前端的火热以及app浪潮势必减退。

超级app的出现

现在的我们一天花的时间最多的是看微信 qq 支付宝 淘宝 京东 王者荣耀 那么我们来分析分析微信能帮我们做什么 微信可以沟通好友 可以查看空间这是最最基础的功能，微信公众号的里面的电信宽带报修，查车票 等等 微信公众号里面的各种博文 微信小程序的头条号 现在微信小程序里的腾讯视屏 还有支付宝我们可以订外卖，找附近的餐馆，支付转账，理财等等基本上需要的服务都可以在这些app里找到 所以说未来的app会向一个超级app发展下去，这个跟家庭场景一样会出现一个类似智能音箱功能的

人工智能的天下

我不会人工智能我也不会机器学习，我甚至连最基本的Python都不会, 但是我要说将来几年最火的应该是大数据，人工智能，机器学习，算法等等
