RN快速上手之ui布局

关于RN搭建环境的坑

关于react-naive的环境搭建具体参考官网的教程，遇到最多的是一个是使用RN初始化项目的时候被墙了导致初始化不能完成，这是解决办法是：

1.使用VPN来通过代理翻墙 具体翻墙自行百度
2.使用镜像源可以通过以下方法设置 

npm config set registry https://registry.npm.taobao.org --global
还有个问题是初始化成功后到编译安装到时候会出错，这是因为我们在初始化到默认是：RN的最高版本所以我们要在初始化项目的时候像下面一样指定安装RN的版本
react-native init myProject --version 0.44.3

快速书写ui

ui的书写其实跟react差不多主要的变化有一下几点

1.React Native中的尺寸都是无单位的，表示的是与设备像素密度无关的逻辑像素点。react的宽度不支持百分比，设置宽度时这样 {width: 10} 具体的屏幕的宽高可以这样

var Dimensions = require('Dimensions');
window.width = Dimensions.get('window').width;
window.height = Dimensions.get('window').height;

2.View组件的默认宽度时100%

3.对于flex布局的支持方法不一样，RN用的不是display：flex这样的写法 

RN用的是flexDiraction：row ／ column 表示子元素是水平一排或者是垂直一列
垂直
<View style={{flexDiration:'cloumn'}}>
    <View>1</View>
    <View>2</View>
    <View>3</View>
</View>    
水平
<View styles={{flexDiration:'row'}}>
    <View>1</View>
    <View>2</View>
    <View>3</View>
</View>        
对于本身元素使用alignItem或者是justifyContent来进行上下左右或者居中的这个跟react里的样式使用是一样的

4.RN里面的定位 RN里面是没有position定位的所以我们需要使用margin值或者是padding值来进行模拟定位。

5.标签的样式都是使用驼峰比如：

<View style={{backgroundColor:'#f0f'}}></View>

总结

如果知道上面的样式差别 就可以直接书写原生app界面ui布局
